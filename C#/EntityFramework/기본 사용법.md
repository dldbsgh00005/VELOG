## 패키지 설치
```
Install-Package Microsoft.EntityFrameworkCore
Install-Package Microsoft.EntityFrameworkCore.Tools
Install-Package Microsoft.EntityFrameworkCore.SqlServer

```

## 자동 코드 생성 
```
Scaffold-DbContext "Server=서버주소;Database=DB명;User Id=유저;Password=비번;" Microsoft.EntityFrameworkCore.SqlServer -OutputDir Models

```

## 생성된 엔티티 사용
```
using (var context = new ExistingDbContext())
{
    var list = context.VisionStatus.Where(v => v.Status == "SUCCESS").ToList();
    // 기존 테이블에 INSERT/UPDATE/SELECT 모두 가능
}

```
