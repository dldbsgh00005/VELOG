# 회고
더 좋은 방법이 있을지 고민했지만 생각이 나지 않아 가장 단순한 방법으로 푼 문제.

# 풀이
N을 1부터 N까지의 수로 나눈다. 나머지가 0인 수는 약수이다. ArrayList에 추가한다. (약수의 개수가 동적이므로 동적 배열에 담았다.)
K번째 즉 ArrayList의 K-1번째 요소를 출력한다.(ArrayList는 Index가 0부터 시작하므로)
```

```
package Algorithm.baekjoon.main;

import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.util.ArrayList;
import java.util.StringTokenizer;

public class Main {

    static int N;
    static int K;

    static ArrayList<Integer> numbers = new ArrayList<>();


    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringTokenizer st = new StringTokenizer(br.readLine());
        N = Integer.parseInt(st.nextToken());
        K = Integer.parseInt(st.nextToken());


        for(int a = 1; a <= N; a++){
            if(N % a == 0) numbers.add(a);
        }

        StringBuilder sb = new StringBuilder();

        if(numbers != null && numbers.size() < K) sb.append(0).append("\n");
        else{
            assert numbers != null;
            sb.append(numbers.get(K-1)).append("\n");
        }
        System.out.println(sb);
    }
}

/**
 * N의 약수들 중 K번째로 작은 수를 출력하라
 *
 * 1 <= N <= 10,000
 * 1 <= K <= N
 *
 * 소수로 나눠질 시 ?배수 - 1
 * **/
