# crontab이란?
```
Unix 계열에서 스크립트 자동화를 위해 사용되는 프로그램인 cron을 기반으로 특정 동작을 자동화(스케줄링) 하는 것이다.
```

## Format
```
[분(0-59)] [시(0-23)] [일(1-31)] [월(1-12)] [요일(0-7)] [명령어(실행시킬 스크립트 경로)] 형태로 작성한다.
```

## Grammmar 
```
* 모든 (모든 분, 시, 일, 월, 요일)
, 복수개 지정 (1,15 => 1분, 15분에 실행)
- 범위 지정 ( 1-5 => 1~5분에 실행)
/ 간격 지정 ( */5 => 5분마다 실행)

```

## Example 
```
ex1) * * * * * /home/usr/test.sh  
=> 매분, 매시간, 매일, 매월, 매요일에 /home/usr/test.sh 를 실행한다. 
=> 매분 /home/usr/test.txt를 실행한다.

ex2) 1,2 1-2 */2 * 0,1,6 /home/usr/test.sh => 매 1,2분 매 1-2시, 2일마다, 모든 월에, 모든 일,월,토요일에 /home/usr/test.sh를 실행한다.
```
